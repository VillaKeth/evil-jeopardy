# Evil Jeopardy 1.2 - Sheldon MVP Project

## Project Overview
Building a Sheldon Cooper-style Mobile Virtual Presence Device (MVP) for Evil Jeopardy 1.2.

**Context:** Switching from nested Evil Jeopardy 2 to Evil Jeopardy 1.2 (normal Jeopardy with evil elements + rocketship building at end).

## Requirements
- **One remote participant** who needs to:
  - Be visible to the room (video feed)
  - Hear everyone in the room clearly
  - Be heard by everyone in the room clearly
  
- **Inspiration:** Sheldon's mobile virtual presence device from The Big Bang Theory (tablet/screen on wheeled platform that can move around)

## Constraints
- **Timeline:** 10 days to procure, build, and set up
- **Tone:** Remote participant can be treated with disdain (evil element!)

## Goals
1. Create a mobile robotic platform with video/audio capability
2. Remote participant can "move around" the room
3. Easy setup and reliable audio/video
4. Integrate with main game flow
